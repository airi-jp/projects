## Setup

To use this with your own GitHub account:

1. Fork and edit the `USER` variable in `index.html` to your GitHub username
2. Push to GitHub
3. Enable GitHub Pages in repository settings (set source to `main` branch)

## Custom Project Metadata

Create a `project.json` file in your project repository with:

```json
{
  "PhotoLink": "https://example.com/photo.png",
  "Description": "Project description"
}
```
